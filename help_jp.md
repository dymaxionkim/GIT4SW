# GIT4SW - SolidWorks Git バージョン管理クライアント

GitワークフローとGit LFSロックを統合し、3D CADモデルの同時編集における競合を防止するエージェンティックCADバージョン管理デスクトップアプリです。

---

## 1. ダッシュボードモード

リポジトリ設定、同期制御、リアルタイムステータスを表示する中央コントロールパネルです。

- **Local Path** — アクティブなローカルGitリポジトリのフルパスを表示します。
- **Change Workspace** — フォルダ選択ダイアログを開き、別のGitリポジトリにワークスペースを切り替えます。
- **Remote Server** — 接続中のリモートURLを表示するか、クローンするURLを入力します。
- **Clone** — リモートリポジトリをローカルパスにクローンします。
- **Active Branch** — 現在のブランチを表示するドロップダウン。別のブランチを選択するとチェックアウトします。
- **Make my branch** — ユーザー名で個人開発者ブランチを作成・切り替え、`main`から分離して作業します。
- **README.md** — README.mdをメモ帳で開きます。保存時に`git add`、`commit`、`push`が自動実行されます。ファイルがなければテンプレートから作成します。
- **Get Latest Version (Sync)** — リモートの変更をpullします。競合検出時は`.backup/`に自動バックアップ後、解決します。最新状態の場合はスキップします。
- **Merge main branch into current branch** — `main`を現在のブランチにマージ、コミット、プッシュ後、元のブランチに復帰します。マージ前に自動バックアップ。既存マージ済みの場合はスキップします。
- **Cleanup LFS Cache** — 未使用の`.git/lfs/objects/`をスキャン・削除するウィザードを開き、ディスク容量を解放します。
- **Auto Sync** — チェックボックス。ON時、起動またはワークスペース変更時に「同期」→「mainマージ」を自動実行します。
- **Live Monitor** — SolidWorks状態(Active/Inactive)、総ファイル数、開かれているファイル、ロック済みファイル、リポジトリサイズをリアルタイム表示します。

---

## 2. ファイルマネージャーモード

ファイルレベルのチェックアウト、チェックイン、およびローカル修正管理機能を提供します。

- **File Table** — ワークスペースのファイルをツリービューで一覧表示（パーツ=緑、アセンブリ=橙、図面=赤、その他=紫）。ステータス、SolidWorks開封状態、ロック状態、所有者を表示。`Ctrl+A`で全選択可能。名前/拡張子/ステータス/SolidWorks/ロック順でソート対応。`.gitignore`準拠。
- **CAD File Preview Canvas** — 選択したCADファイルの180x135サムネイルをサイドバーに表示（OLE+COM抽出）。ファイル1つ選択時のみ有効。
- **Click-to-Copy** — プレビューキャンバスをクリックするとサムネイルビットマップがクリップボード(CF_DIB)にコピーされ、他アプリに貼り付け可能。
- **Refresh** — ファイル一覧、LFSロック、SolidWorksドキュメント状態を即時更新します。
- **Find Top** — COM APIで`.sldasm`依存関係グラフを高速スキャン（ファイル未開封）。最上位アセンブリを赤の**TOP**ラベルで表示します。
- **Open** — ワークスペースフォルダをエクスプローラーで開きます。
- **Lock** — 選択ファイルにLFSロックを取得します。SolidWorksでファイルを開く際も自動取得されます。
- **Unlock File** — 選択ファイルのLFSロックを解除します。
- **Force Unlock** — 他開発者のLFSロックを強制解除します（注意）。
- **Discard** — 選択ファイルを最終コミット状態に戻します。
- **eDrawings** — 選択したCADファイルをeDrawingsで開きます（軽量ビューア）。
- **SolidWorks** — 選択ファイルをSolidWorksで開きます。
- **Diff** — パーツ/図面の2バージョンを比較。コミット履歴ポップアップから選択したコミットを`_THEIRS`、現在を`_OURS`として`.backup/`に抽出し、SolidWorksで開きます。
- **EXPORT** — 図面/パーツ/アセンブリをPDF/DXF/STEP/STEP_ASMに一括変換。非同期バックグラウンド実行。監視タイマー(3分)、UTF-8安全処理、デッドロック防止機能搭載。
- **Version Description** — コミットメッセージを入力するテキストボックス。
- **Upload Selected File Version** — 選択した修正ファイルをコミット、プッシュ、アップロードしロック解除。SolidWorks開封中または他者ロック時はブロック。
- **Upload Every Files Version** — ワークスペース内の全修正ファイルを一括コミット、プッシュ、アップロード、ロック解除。

## 3. 履歴ログモード

過去のリビジョンを参照し、変更履歴を監査します。

- **Revision List** — 過去のコミット一覧（ハッシュ、作成者、日付、メッセージ）。現在のチェックアウト行は緑色太字で強調表示。
- **Graph** — ターミナルを開き、`git log --graph --oneline --all --decorate`でASCIIコミットツリーを表示。

---

## 4. メンテナーモード

プロジェクト管理者向けの統合管理およびリポジトリ初期化モードです。

- **Merge all branches into main** — 全リモートブランチを`main`にマージ、プッシュ後、元の開発者ブランチに復元します。
- **Repository Name + Create New CAD Repository** — リポジトリ名を入力して「Make」をクリックすると、GitHub非公開リポジトリの自動作成、LFS設定、ロック拡張子割り当て、初期コミットプッシュまで一括処理。

---

## 5. 設定モード

Git、SolidWorks、および統合に関するシステムパスを構成します。

- **Path Configurations** — Git、Git LFS、SolidWorks、eDrawings、Git Token、サーバータイプ/URL、デフォルトディレクトリ、GitHub組織名のパス設定。各フィールドに「検索」ボタンがありCドライブをスキャン。`auto_sync`はダッシュボードのチェックボックスで管理。
- **Save Configuration** — 設定を`config.json`に保存し、接続を再初期化します。
- **Edit** — `config.json`をメモ帳で直接開き手動編集。

---

## 6. システムログおよびプロセス終了

操作ログの表示とプロセス制御を行う下部パネルです。

- **Status Indicator** — 待機時は緑「● 待機中」、作業中は赤「● 作業中」を表示。
- **Sequential Button Queuing** — 作業中にボタンをクリックするとキューイングされ、順次実行されます。
- **Terminate** — 実行中のGit操作を強制終了し状態を復元。待機中は無効（ピンク）、作業中は有効（赤）。
- **Clear** — ログウィンドウをクリアします。
